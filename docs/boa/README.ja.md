# BOAガイド（VCDesign Core）

このガイドは、BOAでVCDesignのプロンプトを実行する最短導線を示します。

## クイックスタート
1. `prompts/boa_quickstart.prompt.md` をBOAに読み込む
2. `prompts/value-continuity-protocols_v1.0.md` で設計セッションを実施
3. `prompts/vcdesign_to_testpack.prompt.md` で Test Pack を生成
4. 必要に応じて `prompts/testpack_to_tests.prompt.md` でテストコードを生成
